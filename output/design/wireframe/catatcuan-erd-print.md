# ERD CatatCuan - Versi Print

Gunakan dokumen utama berikut untuk diserahkan atau dicetak:
- `catatcuan-erd-print.html`

Isi HTML tersebut sudah disusun sebagai satu dokumen utuh yang memuat semua bagian ERD dalam format print-friendly.

File `.mmd` tetap disediakan sebagai sumber diagram per bagian jika sewaktu-waktu perlu diedit atau dirender ulang.

Daftar source diagram:
- `catatcuan-erd-print-01-admin-user-warung.mmd`
- `catatcuan-erd-print-02-master-katalog.mmd`
- `catatcuan-erd-print-03-pelanggan-penjualan.mmd`
- `catatcuan-erd-print-04-pengeluaran-hutang.mmd`
- `catatcuan-erd-print-05-kas-laporan.mmd`
