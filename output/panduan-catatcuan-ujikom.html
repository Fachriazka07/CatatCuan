1. Deskripsi Aplikasi

CatatCuan merupakan aplikasi berbasis Android yang dirancang untuk membantu pengguna dalam melakukan pencatatan warung secara lebih cepat, rapi, dan terintegrasi. Aplikasi ini mendukung proses pengelolaan data produk, transaksi penjualan, pelanggan, hutang dan piutang, pengeluaran, buku kas, statistik warung, pembuatan laporan, dan pengaturan printer Bluetooth untuk kebutuhan cetak struk.

Dengan adanya aplikasi ini, proses pencatatan yang sebelumnya dilakukan secara manual dapat dilakukan melalui satu sistem yang lebih efisien dan mudah dipantau. Aplikasi ditujukan terutama untuk pemilik warung yang membutuhkan pengelolaan data harian secara praktis.

Tujuan Aplikasi

  • Membantu pengguna mencatat data warung secara digital.
  • Mempermudah pengelolaan transaksi penjualan dan stok produk.
  • Menyediakan informasi keuangan warung secara lebih cepat dan akurat.
  • Mendukung pembuatan laporan warung berdasarkan periode tertentu.
  • Membantu pencetakan struk transaksi menggunakan printer Bluetooth.

2. Spesifikasi Perangkat

Spesifikasi Minimum Perangkat

Komponen Spesifikasi Minimum
PerangkatSmartphone Android
Memori RAMMinimal 3 GB
PenyimpananMinimal 100 MB ruang kosong
KoneksiInternet aktif untuk sinkronisasi data
BluetoothAktif untuk pencarian dan koneksi printer struk

Spesifikasi yang Disarankan

  • Smartphone Android dengan RAM 4 GB atau lebih.
  • Koneksi internet stabil.
  • Printer thermal Bluetooth ukuran kertas 58 mm.
  • Penyimpanan kosong lebih dari 100 MB agar export laporan dan penyimpanan file tetap berjalan lancar.
Catatan: Berdasarkan implementasi aplikasi, fitur printer Bluetooth saat ini disiapkan untuk platform Android. Aplikasi juga menggunakan internet, notifikasi, Bluetooth, dan lokasi untuk mendukung beberapa fitur utama.

Kebutuhan Sistem Pendukung

  • Akun pengguna yang telah terdaftar di aplikasi.
  • Data warung telah diisi saat onboarding awal.
  • Printer Bluetooth harus dalam kondisi menyala jika akan digunakan untuk cetak struk.

3. Panduan Penggunaan Aplikasi

4.1 Registrasi Akun

Digunakan oleh pengguna baru yang belum memiliki akun.

  1. Buka aplikasi CatatCuan.
  2. Pada halaman awal, pilih menu Daftar.
  3. Masukkan nomor HP.
  4. Masukkan password.
  5. Tekan tombol Daftar Sekarang.
  6. Setelah berhasil, pengguna akan diarahkan ke proses onboarding.
4.2 Login ke Aplikasi

Digunakan oleh pengguna yang sudah memiliki akun.

  1. Buka aplikasi CatatCuan.
  2. Pilih menu Masuk.
  3. Masukkan nomor HP.
  4. Masukkan password.
  5. Tekan tombol Masuk Aplikasi.
  6. Jika data benar, pengguna akan masuk ke halaman utama.
4.3 Onboarding Data Warung

Dilakukan saat pengguna pertama kali masuk dan belum memiliki data warung.

  1. Isi nama pemilik warung.
  2. Isi nama warung.
  3. Isi alamat warung.
  4. Masukkan saldo awal uang laci.
  5. Masukkan modal tambahan jika ada.
  6. Simpan data untuk melanjutkan ke halaman beranda.
4.4 Mengelola Produk

Fitur untuk menambah, melihat, mengubah, dan menghapus produk.

  1. Dari halaman utama, pilih menu Produk.
  2. Tekan tombol +.
  3. Pilih Tambah Produk untuk input manual, atau pilih Scan Produk jika ingin menambahkan barcode lebih cepat.
  4. Periksa atau ubah kode produk apabila diperlukan.
  5. Isi nama produk.
  6. Pilih kategori produk.
  7. Pilih satuan produk.
  8. Masukkan stok produk, atau aktifkan opsi tanpa stok bila produk tidak memakai pencatatan stok.
  9. Masukkan harga modal.
  10. Masukkan harga jual.
  11. Tekan tombol Simpan untuk menyimpan data produk.
  12. Untuk melihat detail produk, pilih salah satu produk pada daftar produk.
  13. Untuk mengubah data produk, buka detail produk lalu edit data yang diperlukan dan simpan kembali.
  14. Untuk menghapus produk, buka detail produk lalu tekan tombol Hapus.
4.5 Melakukan Transaksi Penjualan

Fitur kasir untuk memilih produk, checkout, dan menyimpan transaksi.

  1. Dari halaman utama, pilih menu Transaksi Penjualan.
  2. Cari produk berdasarkan nama atau barcode.
  3. Tambahkan produk ke keranjang.
  4. Atur jumlah barang yang dibeli.
  5. Tekan tombol Checkout.
  6. Masukkan diskon jika ada.
  7. Pilih metode pembayaran Tunai atau Hutang.
  8. Jika tunai, masukkan uang yang diterima.
  9. Jika hutang, pilih pelanggan, isi uang muka jika ada, lalu tentukan jatuh tempo.
  10. Tekan tombol Bayar Sekarang.
  11. Setelah transaksi selesai, pengguna dapat mencetak struk apabila printer Bluetooth sudah diatur.

4. Panduan Penggunaan Lanjutan

5.1 Mengelola Pelanggan
  1. Buka menu Pelanggan.
  2. Tekan tombol +.
  3. Isi nama pelanggan.
  4. Isi alamat dan nomor telepon bila diperlukan.
  5. Tekan tombol Simpan.
  6. Untuk melihat detail pelanggan, pilih data pelanggan pada daftar.
5.2 Mengelola Hutang dan Piutang
  1. Buka menu Hutang & Piutang.
  2. Pilih tab Hutang, Piutang, atau Lunas.
  3. Tekan tombol + untuk menambah data baru.
  4. Pilih jenis transaksi, yaitu Saya Berhutang atau Orang Berhutang.
  5. Pilih pelanggan.
  6. Masukkan catatan jika diperlukan.
  7. Masukkan nominal hutang atau piutang.
  8. Atur tanggal jatuh tempo jika diperlukan.
  9. Simpan data.
  10. Untuk melihat rincian dan progres pembayaran, buka data pada daftar.
5.3 Mengelola Pengeluaran
  1. Buka menu Pengeluaran.
  2. Tekan tombol +.
  3. Pilih kategori pengeluaran.
  4. Masukkan nominal pengeluaran.
  5. Isi keterangan pengeluaran.
  6. Simpan data.
  7. Data akan masuk ke daftar pengeluaran dan ikut dihitung pada laporan.
5.4 Mengelola Buku Kas
  1. Buka menu Buku Kas.
  2. Lihat riwayat mutasi kas yang sudah tercatat.
  3. Gunakan filter tanggal untuk menampilkan data pada periode tertentu.
  4. Tekan tombol + untuk menambah data kas baru.
  5. Pilih jenis transaksi kas, yaitu Uang Masuk, Uang Keluar, Transfer Saldo, atau Penyesuaian Saldo.
  6. Isi data transaksi sesuai kebutuhan.
  7. Simpan data agar saldo kas diperbarui.
5.5 Melihat Laporan
  1. Buka menu Laporan.
  2. Tentukan periode laporan, seperti hari ini, minggu ini, bulan ini, triwulan, atau custom.
  3. Pilih jenis laporan:
    • Ringkasan Keuangan
    • Laporan Penjualan
    • Laporan Pengeluaran
    • Laporan Buku Kas
    • Laporan Hutang/Piutang
  4. Lihat ringkasan data yang ditampilkan aplikasi.
  5. Gunakan tombol export untuk membagikan laporan dalam bentuk PDF atau Excel.
5.6 Melihat Statistik Warung
  1. Buka menu Stats.
  2. Pilih periode statistik, seperti harian, mingguan, bulanan, atau triwulanan.
  3. Lihat data omzet, profit, pengeluaran, dan arus kas bersih.
  4. Lihat grafik tren penjualan, arus kas, insight cepat, dan produk terlaris.
5.7 Mengatur Printer Bluetooth
  1. Buka menu Pengaturan.
  2. Pilih menu Printer Bluetooth.
  3. Tekan tombol Cari Printer.
  4. Pilih salah satu printer yang terdeteksi.
  5. Jika printer tidak muncul, gunakan opsi Input Manual.
  6. Simpan printer default.
  7. Tekan tombol Tes Printer untuk memastikan printer berfungsi.
5.8 Pengaturan Lainnya

Aplikasi juga menyediakan menu pengaturan tambahan yang dapat digunakan untuk:

  • Mengelola profil pengguna.
  • Mengelola data warung.
  • Mengubah password.
  • Mengatur periode default.
  • Mengelola notifikasi.
  • Mengelola kategori produk.
  • Mengelola kategori pengeluaran.
  • Mengelola satuan produk.
  • Mengatur saldo awal.
  • Melihat informasi aplikasi.
  • Keluar dari aplikasi.

5. Panduan Penggunaan Admin

Selain digunakan oleh pemilik warung melalui aplikasi mobile, sistem CatatCuan juga memiliki panel admin berbasis web. Panel admin digunakan untuk memantau data pengguna, warung, master data, konfigurasi aplikasi, serta pengujian notifikasi sistem.

5.1 Login ke Panel Admin

Digunakan oleh administrator untuk masuk ke dashboard web admin.

  1. Buka halaman login admin CatatCuan melalui browser.
  2. Masukkan email admin.
  3. Masukkan password admin.
  4. Tekan tombol Masuk.
  5. Jika data benar, admin akan diarahkan ke halaman dashboard.
5.2 Melihat Dashboard Admin

Dashboard menampilkan ringkasan kondisi platform CatatCuan.

  1. Setelah login, admin masuk ke menu Dashboard.
  2. Lihat total pengguna yang terdaftar.
  3. Lihat total warung yang sudah dibuat.
  4. Lihat total volume transaksi global.
  5. Lihat status sistem.
  6. Amati grafik pertumbuhan pengguna dan warung.
  7. Amati distribusi master data dan status user.
  8. Lihat log aktivitas terbaru pada sistem.
5.3 Monitoring Users dan Warung

Menu ini digunakan untuk memantau akun user dan data warung yang terhubung ke sistem.

  1. Buka menu Users & Warung.
  2. Gunakan tab User untuk melihat daftar akun pengguna.
  3. Gunakan tab Warung untuk melihat daftar warung.
  4. Gunakan kolom pencarian untuk mencari user, nomor HP, nama warung, atau nama pemilik.
  5. Gunakan filter status pada daftar user jika diperlukan.
  6. Pilih salah satu user untuk membuka detail akun.
  7. Pilih salah satu warung untuk membuka detail warung.
5.4 Mengelola Status User

Admin dapat mengubah status akun user sesuai kebutuhan.

  1. Buka detail user dari menu Users & Warung.
  2. Pada bagian kontrol akun, pilih status yang diinginkan.
  3. Pilih Aktifkan jika user boleh login dan menggunakan aplikasi mobile.
  4. Pilih Nonaktifkan jika akses ingin dihentikan sementara.
  5. Pilih Blokir jika akun harus ditolak aksesnya karena alasan tertentu.
  6. Setelah status diubah, sistem akan menyimpan perubahan dan memperbarui data user.
5.5 Melihat Detail User dan Detail Warung

Admin dapat melihat informasi operasional secara lebih rinci.

  1. Pada detail user, admin dapat melihat profil akun, status, jumlah warung, jumlah produk, pelanggan, transaksi, pengeluaran, dan saldo awal.
  2. Admin juga dapat melihat penjualan terbaru dan pengeluaran terbaru milik user tersebut.
  3. Pada detail warung, admin dapat melihat identitas warung, pemilik, alamat, nomor telepon, uang kas, kas operasional, serta statistik aktivitas warung.
  4. Admin juga dapat melihat transaksi penjualan terbaru, pengeluaran terbaru, dan entri buku kas terbaru pada warung tersebut.
5.6 Mengelola Master Kategori Produk

Master kategori dipakai sebagai data default untuk warung baru.

  1. Buka menu Master Kategori.
  2. Tekan tombol Tambah Kategori untuk membuat kategori baru.
  3. Isi nama kategori.
  4. Pilih icon kategori.
  5. Simpan data kategori.
  6. Untuk mengubah kategori, tekan tombol edit pada baris kategori yang dipilih.
  7. Untuk menonaktifkan kategori, tekan tombol hapus/nonaktifkan.
  8. Admin juga dapat mengubah status aktif dan nonaktif kategori.
5.7 Mengelola Master Pengeluaran

Menu ini digunakan untuk mengatur kategori pengeluaran global.

  1. Buka menu Master Pengeluaran.
  2. Tekan tombol Tambah Kategori.
  3. Isi nama kategori pengeluaran.
  4. Pilih tipe kategori, yaitu Bisnis atau Pribadi.
  5. Pilih icon kategori.
  6. Simpan data.
  7. Admin dapat mengedit, menonaktifkan, atau mengaktifkan kembali kategori sesuai kebutuhan.
5.8 Mengelola Master Satuan

Master satuan digunakan sebagai pilihan satuan produk di aplikasi mobile.

  1. Buka menu Master Satuan.
  2. Tekan tombol Tambah Satuan.
  3. Masukkan nama satuan, misalnya PCS, KG, BOTOL, atau PACK.
  4. Simpan data satuan.
  5. Admin dapat mengedit satuan yang sudah ada.
  6. Admin dapat menonaktifkan atau mengaktifkan kembali satuan sesuai kebutuhan.
5.9 Mengelola Konfigurasi Aplikasi dan Test Notifikasi

Menu konfigurasi dipakai untuk mengatur parameter global dan menguji notifikasi sistem.

  1. Buka menu Konfigurasi.
  2. Pada bagian App Config, admin dapat menambah konfigurasi baru.
  3. Isi key, value, dan deskripsi konfigurasi.
  4. Simpan konfigurasi agar berlaku secara global.
  5. Admin dapat mengedit atau menghapus konfigurasi yang sudah ada.
  6. Pada bagian Test Notifikasi, admin dapat menjalankan push notification manual.
  7. Admin juga dapat menguji reminder hutang jatuh tempo, alert stok menipis, dan pengingat pencatatan harian.
5.10 Logout dari Panel Admin
  1. Setelah selesai menggunakan panel admin, pilih tombol Logout pada sidebar.
  2. Sistem akan mengeluarkan admin dari sesi login.
  3. Admin akan kembali ke halaman login.

6. Penutup

Panduan ini disusun untuk menjelaskan cara instalasi dan penggunaan aplikasi CatatCuan secara singkat, runtut, dan mudah dipahami. Melalui aplikasi ini, pengguna dapat melakukan pencatatan warung mulai dari pengelolaan produk, transaksi penjualan, pengeluaran, kas, hutang piutang, sampai pembuatan laporan secara digital.

Diharapkan aplikasi CatatCuan dapat membantu pengguna dalam meningkatkan ketertiban pencatatan warung, mempercepat proses operasional harian, dan mempermudah pengambilan keputusan berdasarkan data yang tersedia.
